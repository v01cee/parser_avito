# Руководство по использованию прокси

## Безопасная частота запросов

### С прокси (Residential/Datacenter):
- **Интервал 1 минута** - ✅ Безопасно
- **Интервал 30 секунд** - ⚠️ Возможно, но рискованно
- **Интервал 15 секунд** - ❌ Высокий риск блокировки

### Без прокси (серверный IP):
- **Интервал 5-10 минут** - ✅ Безопасно
- **Интервал 1 минута** - ⚠️ Возможны блокировки
- **Интервал меньше 1 минуты** - ❌ Высокий риск блокировки

## Рекомендации

### Для интервала 1 минута (как требуется):
1. **Обязательно используй прокси** - без него будут блокировки
2. **Residential прокси** - лучше всего (выглядят как домашние IP)
3. **Datacenter прокси** - работают, но могут быть заблокированы быстрее
4. **Ротация прокси** - если есть несколько прокси, можно ротировать

### Настройка прокси:

1. Добавь в `.env` файл:
```bash
AVITO_PROXY=http://user:pass@proxy.example.com:8080
# или
AVITO_PROXY=socks5://proxy.example.com:1080
```

2. Перезапусти контейнер:
```bash
docker-compose restart
```

## Где взять прокси:

### Бесплатные (нестабильные):
- https://free-proxy-list.net/
- https://www.proxy-list.download/

### Платные (рекомендуется):
- **Bright Data** (Luminati) - residential прокси, ~$500/мес
- **Oxylabs** - residential прокси, ~$300/мес
- **Smartproxy** - residential прокси, ~$75/мес
- **Proxy-Cheap** - datacenter прокси, ~$20/мес

### Для теста:
- Можно использовать бесплатные прокси, но они часто не работают
- Лучше купить дешевый datacenter прокси для начала (~$5-10/мес)

## Важно:

- **1 запрос в минуту с прокси** - безопасно
- **Без прокси** - будут блокировки даже с интервалом 1 минута
- **Residential прокси** - самый надежный вариант
- **Ротация IP** - если прокси поддерживает, используй
